# Registration
Simple Registration and write login and password to text file 

## How to build:
1. Clone this repository
2. Execute ```make``` for build
3. Execute ```make run``` for run project


## Database.txt example:
|Login       |Password       |
|------------|:-------------:|
|User1       |User1Password  |
|User2       |User2Password  |
|User3       |User3Password  |
